< Government of Saint Lucia Website (govt.lc) Redesign >

< Overview >

This redesign project was created as a personal initiative to improve the official Government of Saint Lucia website. Motivated by my connection to the island where I grew up, the goal was to create a more modern, accessible, and user-friendly experience while preserving all existing content and navigation structure.

< Project Goals >

  • Improve the user experience for Saint Lucian citizens accessing government services
  • Apply UX design principles to a high-impact, real-world platform
  • Build a scalable, component-based foundation for future digital transformation within the public sector
  • Give back to my country by modernizing a site that directly serves the community

< Features >

Fully redesigned interface with a sleek, modern visual style
Reorganized content structure for better information architecture and usability
Persistent header navigation and new footer with quick links, contact info, and disclaimers
Individual page layouts for key sections like services, news, and “Popular Topics”

< Future Plans >

  • Build the full site using a modern frontend framework like React or Vue
  • Implement backend functionality (e.g., CMS integration or dynamic content handling)
  • Add improved search functionality and breadcrumb navigation
  • Conduct user testing and accessibility audits prior to deployment
